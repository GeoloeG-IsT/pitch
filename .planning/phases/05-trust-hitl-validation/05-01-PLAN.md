    2. Create `apps/api/app/services/confidence.py` with four functions:
       - `compute_retrieval_signal(chunks: list[dict]) -> float`: Average top-3 `relevance_score` (falling back to `similarity` key). Returns 0.0 for empty list.
       - `compute_coverage_signal(question: str, chunks: list[dict], document_titles: dict[str, str]) -> float`: Regex extract words (3+ chars), count matches in chunk content + doc titles. Return ratio. Return 0.5 for very short questions.
       - `compute_confidence_score(retrieval: float, llm: float, coverage: float) -> tuple[float, str]`: Weights: retrieval 0.40, LLM 0.35, coverage 0.25. Multiply by 100, clamp 0-100, round to 1 decimal. Tier: >=70 "high", >=40 "moderate", <40 "low".
       - `extract_llm_confidence(full_response: str) -> tuple[str, float]`: Regex match `\nCONFIDENCE:\s*([0-9]*\.?[0-9]+)\s*$`. Strip the line from response. Default 0.5 if no match. Clamp 0-1.
