    All components are `"use client"` and import from `@/lib/utils` for `cn()`.

    1. **text-section.tsx** — Renders chunk_type="text" content:
       - Import `ReactMarkdown` from `react-markdown` and `remarkGfm` from `remark-gfm`
       - Accept props: `{ content: string }`
       - Render: `<div className="prose prose-neutral max-w-none"><ReactMarkdown remarkPlugins={[remarkGfm]}>{content}</ReactMarkdown></div>`
       - The `prose` class comes from @tailwindcss/typography (installed in Plan 01)
       - `max-w-none` because the Card container constrains width

    2. **metric-card.tsx** — Single key metric display used within TableSection:
       - Accept props: `{ label: string; value: string; subLabel?: string }`
       - Render a div with:
         - Label: `<p className="text-sm text-muted-foreground">{label}</p>` (14px per UI-SPEC Label role)
         - Value: `<p className="text-[32px] font-semibold leading-tight text-foreground">{value}</p>` (32px/600 per UI-SPEC Display role)
         - SubLabel (optional): `<p className="text-sm text-muted-foreground mt-1">{subLabel}</p>`
       - Container: `className="p-4 rounded-lg bg-muted/50"`

    3. **table-section.tsx** — Renders chunk_type="table" content as metric grid:
       - Import `parseTableContent` from `@/lib/parse-table-content` and `MetricCard`
       - Accept props: `{ content: string }`
       - Call `parseTableContent(content)` to get MetricPair array
       - If pairs returned: render a responsive grid of MetricCard components
         - Grid classes: `grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4`
         - Mobile stacks vertically (1-col) per user decision
       - If null (unparseable): fall back to `<div className="prose prose-neutral max-w-none"><ReactMarkdown remarkPlugins={[remarkGfm]}>{content}</ReactMarkdown></div>` which renders the raw markdown table styled via typography plugin

    4. **heading-section.tsx** — Renders chunk_type="heading" as section divider:
       - Accept props: `{ content: string; id: string }`
       - NOT inside a Card (per UI-SPEC: "Render as a section divider, not inside a Card")
       - Render:
         ```tsx
         <div id={`section-${id}`} className="mt-12 mb-4">
           <h2 className="text-2xl font-semibold leading-tight text-foreground">{content}</h2>
           <Separator className="mt-3" />
         </div>
         ```
       - 48px top margin (mt-12 = 3rem ~ 48px) per UI-SPEC 2xl spacing

    5. **image-caption-section.tsx** — Renders chunk_type="image_caption":
       - Import `Image` icon from `lucide-react`
       - Accept props: `{ content: string }`
       - Render inside a Card with `bg-muted` background:
         ```tsx
         <Card className="bg-muted">
           <CardContent className="p-4 flex items-start gap-2">
             <Image className="h-4 w-4 text-muted-foreground mt-0.5 shrink-0" />
             <p className="text-sm italic text-muted-foreground">{content}</p>
           </CardContent>
         </Card>
         ```

    6. **section-card.tsx** — Dispatcher component that wraps content in Card or delegates to HeadingSection:
       - Accept props: `{ chunk: PitchChunk; onInView?: (id: string, inView: boolean) => void }`
       - Import `useInView` from `react-intersection-observer`
       - Set up intersection observer: `const { ref } = useInView({ threshold: 0.3, onChange: (inView) => onInView?.(chunk.id, inView) })`
       - For `chunk_type === "heading"`: render `<HeadingSection>` directly (no Card wrapper), attach `ref` to wrapper div
       - For `chunk_type === "text"`: render `<Card><CardContent className="p-6 lg:p-8"><TextSection content={chunk.content} /></CardContent></Card>`
       - For `chunk_type === "table"`: render `<Card><CardContent className="p-6 lg:p-8"><TableSection content={chunk.content} /></CardContent></Card>`
       - For `chunk_type === "image_caption"`: render `<ImageCaptionSection content={chunk.content} />` (it has its own Card)
       - All Card instances: `className="bg-card"` (uniform white cards per user decision)
       - Attach `ref` and `id={`section-${chunk.id}`}` to the outermost element of each case

    7. **document-group.tsx** — Groups sections by source document:
       - Accept props: `{ document: PitchDocument; onSectionInView?: (id: string, inView: boolean) => void }`
       - Render document title as Display-size heading: `<h2 className="text-[32px] font-semibold leading-tight text-foreground mb-6">{document.title}</h2>`
       - Map `document.chunks` to `<SectionCard>` components with `gap-8` (32px = xl spacing) between cards
       - Container: `<div className="mb-12">` (48px = 2xl spacing between document groups)
