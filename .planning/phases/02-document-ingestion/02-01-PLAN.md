**1. Create `apps/api/app/services/parsers/pdf_parser.py`:**

Use pymupdf4llm with `page_chunks=True`. Each page becomes one Section where `section_number = page_number` (1-based). Tables on pages are detected and chunk_type set to "table" if tables present, otherwise "text". Metadata includes toc_items and table_count.

```python
import pymupdf4llm
import pymupdf
from app.models.document import Section

def parse_pdf(file_bytes: bytes) -> list[Section]:
    doc = pymupdf.open(stream=file_bytes, filetype="pdf")
    pages = pymupdf4llm.to_markdown(doc, page_chunks=True)
    doc.close()

    sections = []
    for page_data in pages:
        page_num = page_data["metadata"]["page"]  # 1-based page number
        text = page_data["text"].strip()
        if not text:
            continue

        tables = page_data.get("tables", [])
        has_tables = len(tables) > 0

        sections.append(Section(
            content=text,
            page_number=page_num,
            section_number=page_num,
            chunk_type="table" if has_tables else "text",
            metadata={
                "toc_items": page_data.get("toc_items", []),
                "table_count": len(tables),
            },
        ))
    return sections
```

NOTE: pymupdf4llm page_chunks metadata key might be "page" instead of "page_number" depending on version. Check the actual output structure. Use `page_data["metadata"]["page"]` and adjust if needed. The implementation must handle both possible keys defensively:
```python
meta = page_data.get("metadata", {})
page_num = meta.get("page", meta.get("page_number", idx + 1))
```

**3. Create `apps/api/app/services/parsers/excel_parser.py`:**

CRITICAL: The demo financial model (`content/output/financial-model.xlsx`) is generated by openpyxl build scripts that write Excel formulas (e.g., `=B4*B5`). When read with `data_only=True`, these formula cells return None because openpyxl never evaluates formulas -- it only reads cached values, and openpyxl-generated files have no cached values.

Solution: Try `data_only=True` first. If a cell value is None but the cell had a formula, try re-reading with `data_only=False` to get the formula string as fallback. Implement a two-pass approach:
1. Open with `data_only=True` to get evaluated values where available
2. Open with `data_only=False` as fallback for cells that returned None
3. For cells where both return None, use empty string

**4. Create `apps/api/app/services/chunker.py`:**

Structure-aware sub-chunking using tiktoken for token counting. Parameters: `max_tokens=500`, `overlap_tokens=50`.

For sections under max_tokens: pass through unchanged.
For sections over max_tokens: split by sentences/paragraphs first, then by words if needed. Use the `_split_with_overlap` helper that:
1. Splits content into sentences (split on `. ` or newlines)
2. Accumulates sentences until approaching max_tokens
3. Creates sub-chunk, then starts next sub-chunk by backing up overlap_tokens worth of text
4. Each sub-chunk preserves parent's page_number, section_number, chunk_type
5. Sub-chunk metadata adds: sub_chunk (int index), parent_section (int)

```python
import tiktoken
from openai import OpenAI
from app.models.document import Section, EmbeddedChunk
from app.core.config import settings

_encoding = tiktoken.encoding_for_model("text-embedding-3-small")

def count_tokens(text: str) -> int:
    return len(_encoding.encode(text))

async def embed_chunks(chunks: list[Section]) -> list[EmbeddedChunk]:
    client = OpenAI(api_key=settings.openai_api_key)

        batch = chunks[i:i + batch_size]
        texts = [c.content for c in batch]

        response = client.embeddings.create(
            input=texts,
            model="text-embedding-3-small",
        )

        for chunk, emb_data in zip(batch, response.data):
            assert len(emb_data.embedding) == 1536, f"Expected 1536 dimensions, got {len(emb_data.embedding)}"
            results.append(EmbeddedChunk(
                content=chunk.content,
                page_number=chunk.page_number,
                section_number=chunk.section_number,
                chunk_type=chunk.chunk_type,
                metadata=chunk.metadata,
                embedding=emb_data.embedding,
                token_count=count_tokens(chunk.content),
            ))
    return results
```
