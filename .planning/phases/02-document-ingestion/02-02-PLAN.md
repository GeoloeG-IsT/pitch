1. Create `apps/api/app/services/ingestion.py`:
   - Import `process_file` from `app.services.pipeline`
   - Import `get_service_client` from `app.core.supabase`
   - Import `logging`; create logger `logger = logging.getLogger(__name__)`

   - Function `async def process_document(doc_id: str, file_bytes: bytes, file_type: str) -> None`:
     - Get service client: `client = get_service_client()`
     - Update document status to "processing": `client.table("documents").update({"status": "processing"}).eq("id", doc_id).execute()`
     - Try:
       - `chunks = await process_file(file_bytes, file_type, doc_id)`
       - Batch insert chunks (insert in groups of 50 to avoid payload limits):
         ```python
         batch_size = 50
         for i in range(0, len(chunks), batch_size):
             batch = chunks[i:i + batch_size]
             client.table("chunks").insert(batch).execute()
         ```
       - Update document status to "ready" and set metadata with chunk_count:
         `client.table("documents").update({"status": "ready", "metadata": {"chunk_count": len(chunks)}}).eq("id", doc_id).execute()`
     - Except Exception as e:
       - Log error: `logger.exception(f"Failed to process document {doc_id}")`
       - Update document status to "error": `client.table("documents").update({"status": "error", "metadata": {"error": str(e)}}).eq("id", doc_id).execute()`

   - Function `async def delete_document_chunks(doc_id: str) -> None`:
     - `client = get_service_client()`
     - `client.table("chunks").delete().eq("document_id", doc_id).execute()`

   - Use a hardcoded `DEMO_USER_ID` constant (UUID string) for user_id until Phase 6 adds auth. Comment: `# TODO(Phase 6): Replace with authenticated user from request`
   - Helper `_detect_file_type(filename: str) -> str`: extract extension, map to valid file_type values. Raise HTTPException(400) for unsupported types.

   **POST /documents** (upload):
   - Accept `UploadFile = File(...)` and optional `title: str = Form(None)` (defaults to filename without extension)
   - `file_bytes = await file.read()` -- MUST read bytes before background task (UploadFile closes after endpoint returns)
   - Detect file_type from filename
   - Insert document record via service client: `client.table("documents").insert({...}).execute()`. Fields: user_id=DEMO_USER_ID, title, file_name=file.filename, file_type, file_size_bytes=len(file_bytes), status="pending"
   - Extract document id from insert response
   - `background_tasks.add_task(process_document, doc_id, file_bytes, file_type)`
   - Return 201 with document record

   **GET /documents** (list):
   - Query documents table ordered by created_at desc: `client.table("documents").select("*").order("created_at", desc=True).execute()`
   - For each document, get chunk count: `client.table("chunks").select("id", count="exact").eq("document_id", doc.id).execute()`
   - Return DocumentListResponse

   **GET /documents/{doc_id}** (get single):
   - Query single document: `client.table("documents").select("*").eq("id", doc_id).single().execute()`
   - Get chunk count
   - Return DocumentResponse or 404

   **DELETE /documents/{doc_id}**:
   - Verify document exists (404 if not)
   - Delete document (chunks cascade due to FK ON DELETE CASCADE)
   - Return 204 No Content

   **PUT /documents/{doc_id}** (re-upload / replace):
   - Accept `UploadFile = File(...)`
   - Verify document exists (404 if not)
   - `file_bytes = await file.read()`
   - Delete old chunks: `await delete_document_chunks(doc_id)`
   - Update document: status="pending", file_name, file_size_bytes, file_type
   - `background_tasks.add_task(process_document, doc_id, file_bytes, file_type)`
   - Return 200 with updated document
